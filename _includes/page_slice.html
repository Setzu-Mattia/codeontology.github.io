Ontology

CodeOntology is an ontology to define source code: flexibility, ease of use and a strict layer structure make it possible to define more languages.

Extraction framework

CodeOntology is an extraction tool to parse Java source code and generate datasets: SPARQL queries over source code reach the gap in search expressivity. An ontology born to create ontologies.

Datasets

CodeOntology is a ever-growing set of dataset generated by the extraction framework.
